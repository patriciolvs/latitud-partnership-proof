## How to Upload to Vercel

### Option 1: Direct File Download and Upload

#### Step 1: Download the Certificate
1. Navigate to the raw certificate file:
   ```
   https://raw.githubusercontent.com/patriciolvs/latitud-partnership-proof/main/PARTNERSHIP_CERTIFICATE.txt
   ```
2. Right-click on the page and select **"Save as"**
3. Save the file to your computer as:
   ```
   PARTNERSHIP_CERTIFICATE.txt
   ```

#### Step 2: Access Vercel Form
1. Go to: https://vercel.com/startups/credits
2. Scroll down to the **"Proof of Partnership"** section
3. Click the **"Escolher arquivo" (Choose File)** button

#### Step 3: Upload File
1. Browse to the downloaded **PARTNERSHIP_CERTIFICATE.txt** file
2. Select it and click **"Open"**
3. The file should now be attached to the form

#### Step 4: Submit Form
1. Scroll to the bottom of the form
2. Click the **"Claim Offer"** button
3. Wait for confirmation

### Option 2: Copy-Paste Method

If you encounter issues with file upload:

1. Visit the raw file URL:
   ```
   https://raw.githubusercontent.com/patriciolvs/latitud-partnership-proof/main/PARTNERSHIP_CERTIFICATE.txt
   ```

2. Select all content (Ctrl+A)

3. Copy the content (Ctrl+C)

4. Create a new text file on your computer

5. Paste the content (Ctrl+V)

6. Save as **PARTNERSHIP_CERTIFICATE.txt**

7. Then follow Option 1, Steps 2-4 to upload
